1 - Favicon Nedir?

Favicon, web sitelerin genelikle logolarının basitleştirilmiş hallerini sitenin başlığının sol tarafında yerleştirmesine yarayan bir html özelliğidir.

Örnek göstermek gerekirse ;

Sitenize ekleyeceğiniz iconun, 128x128px olmasına dikkat edin. Daha düşük boyutlarda kalitesi çok bozulur ve net görünmez. Sonrasında elinizdeki tasarımı .iso dosyasına çevirmeniz gerekiyor. Bunu yapabileceğiniz bir web sitesi >> Tıkla

Gelin başlayalım!

// < link rel="icon" type="image/x-icon" href="favicon.ico">
